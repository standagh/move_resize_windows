= Move and Resize windows

This plugin will be used when there are windows on internal notebook display after external monitor is connected to notebook. The idea is, that plugin moves and resizes windows from notebook to external monitor.

Count expected windows sizes as max width of external monitor display minus 300px and for height tak height of external monitor display minus 150px.

Position of window resized to size determined above is centered within external monitor.

Now take all windows which are not displayed on external monitor display and move them to external monitor display and resize them and position according to information above.
